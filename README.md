## Build Setup（启动项目）

```
#git clone
git clone https://github.com/front-sjkx/swebsite.git

# install dependencies(下载依赖)
npm install

# serve will load at localhost: 3000(项目启动)
npm start 

# build for production with minification（打包生产环境）
npm run build
```
